
🗺️ Сторінка: Мапа розташування
Щоб знайти скульптурку, введіть її назву в поле пошуку.
Оберіть потрібну зі списку результатів.
Для перегляду інформації про скульптурку натисніть на її маркер на мапі.
Доступна можливість змінити вид мапи (наприклад, супутниковий або стандартний).

📋 Сторінка: Список скульптурок
Скористайтеся пошуком, ввівши назву скульптурки – список оновиться автоматично.
Є можливість сортування за різними критеріями.
Відображається загальна кількість скульптур.
Щоб переглянути лише улюблені, натисніть кнопку у вигляді ❤️ (натискання ще раз вимикає фільтр).
Для перегляду детальної інформації натисніть на картку скульптурки.

🗿 Сторінка: Перегляд скульптурки
Відображається повна інформація: фото, геолокація, рейтинг, відгуки.
Доступний перегляд 3D-моделі.
Оцінювання: натисніть потрібну кількість зірок (1–5), щоб поставити оцінку.
Напишіть свій відгук у відповідному полі.
Щоб додати до улюблених, натисніть ❤️.
Для повернення назад – натисніть кнопку зі стрілкою.

✉️ Сторінка: Зворотній зв’язок
Оберіть категорію запиту.
Введіть повідомлення у відповідне поле.
Натисніть кнопку надсилання.

🧩 Сторінка: Квести
Відображається дерево квестів з прогресом.
Квести потрібно виконувати послідовно.
Додаткова інформація з’являється при наведенні на квест.
Дерево можна переміщати та змінювати масштаб.
Для центрування натисніть кнопку у вигляді перехрестя стрілок.

🏆 Сторінка: Досягнення
Відображаються всі досягнення користувача.
Є фільтр для перегляду:
Всі
Виконані
Невиконані

👤 Сторінка: Особистий кабінет
Переглядайте залишені відгуки та історію дій.
Доступне сортування записів.
Для видалення окремого запису натисніть кнопку 🗑️ на відповідному елементі.
Для очищення всієї історії натисніть спеціальну кнопку.
Можна змінити ім’я користувача.

Для зміни пароля:
Введіть старий та новий пароль.
На вашу електронну адресу надійде код підтвердження.
Введіть код у відповідному полі.
Для зміни аватара натисніть на нього та оберіть новий файл із пристрою.

🎉 Сторінка: Події
Відображаються актуальні події, пов’язані зі скульптурками.
Є можливість сортування (від старіших до новіших).
Для пошуку введіть ключове слово – список подій оновиться автоматично.
